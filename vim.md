# vim operation
===============================
1. command1: :ls
// 展示出现在vim的缓冲区
//配套指令1: :b<缓冲区值> // 直达这个缓冲区
//配套指令2: :bd <缓冲区值> // 删除该缓冲区

2. 操作2:分屏
//(1)竖直分屏: :vs   //配套：分屏时打开指定文件: :vs/sp filename.md
//(2)水平分屏: :sp   //配套：窗口导航和切换: 移动方向就是hjkl,不过要按住ctrl+w
//(3)输出命令到新窗口
//基础方法：使用:new + :r !命令
{
    " 1. 创建新窗口并执行命令
:new | r !ls -la          " 水平分屏显示 ls 结果
:vnew | r !pwd            " 垂直分屏显示 pwd 结果

" 2. 分步操作
:new                     " 创建新窗口
:r !date                 " 插入 date 命令结果
:r !echo "Hello World"   " 插入 echo 结果
}
