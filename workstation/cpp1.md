# type
- 和c一样的类型(如int,double,void)
1. typedef给类型赋予别名
2. 确定某个类型的取值范围：
```cpp
#include <limits>

//std::numeric_limits<类型>::max()或min()求最大值，最小值
unsigned long long a = std::numeric_limits<unsigned long long int>::max();
cout << a << endl;
```
- unsigned在移位操作中很重要
# 变量
## 两个概念
- 生存期：分配和销毁内存时机
- 作用域：变量可见的位置
## 变量类型
- 全局变量
1. 全局变量可以初始化
2. 全局变量可在多个文件间共享
3. 分为静态全局和全局
4. 在函数外的变量不能执行表达式，只能用于声明变量
5. 同名时局部变量覆盖全局变量       //所以全局变量取名特殊点
## 复制
## 交换逻辑
- 声明t用于交换a,b的值
// 大括号：局部作用域,在作用域内声明的变量仅仅在作用域内有用
// 习惯：尽量缩小变量的作用域，某些变量值用一次就用一个大括号
eg
```cpp
int a = 5, b = 8;
{
int t = a;  // t的作用域只在大括号内
a = b;
b = t;
}
cout << a << "," << b << endl;
```
# 常量
## 枚举类型
- 关键词:enum
语法：
```cpp
enum 名称{ele1, ele2, ....};
//或者用typedef简化
typedef enum （名称){ele1,...} 名称;
```
# 类型修饰符
## 种类
### 类型修饰符
- const 常量
- volatile 不要进行自动优化
### 存储修饰符
- register 寄存器,已弃用
- static 静态变量或 //静态区独立于栈和堆
- extern 全局变量引用
